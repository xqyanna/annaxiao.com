# Anna Xiao · 个人网站

黑发眼镜女孩的个人主页，温暖文艺风。

## 快速开始

直接在浏览器打开 `index.html` 即可预览。

## 页面导航

- `index.html` → 主页
- `journey.html` → 成长历程
- `projects.html` → 实习 / 项目
- `interests.html` → 爱好
- `gallery.html` → 摄影相册
- `games.html` → 游戏经历

## 技术栈

- HTML / CSS / JavaScript（原生）
- Tailwind CSS（CDN）
- Google Fonts：Newsreader + Plus Jakarta Sans
- 无需构建，无需 npm，打开即用
